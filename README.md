# VK Mini App для магазина Kotleta Boost (PUBG)

Мини‑приложение ВКонтакте для оформления заказов на буст/сопровождение в PUBG. Проект включает веб‑интерфейс VK Mini Apps и бэкенд на Go с PostgreSQL. Сейчас репозиторий содержит только документацию.

## Статус
- MVP работает
- Ведется доработка UX и админ‑панели

## Функциональность
**Покупатель**
- Выбор режима: Metro / Classic
- Создание заказа и переход в чат
- Подтверждение оплаты (ручное)
- Подтверждение завершения работы

**Исполнитель**
- Биржа заказов
- Взятие заказа в работу
- Чат с заказчиком
- Подтверждение оплаты и завершения

**Админ**
- Сводная статистика
- Просмотр заказов
- Просмотр исполнителей
- Доступ к чатам (read‑only)

## Технологии
- Frontend: React + VKUI + Vite
- Backend: Go
- DB: PostgreSQL (Supabase)
- Hosting: VPS + Nginx

## Архитектура (кратко)
- VK Mini App (статичный фронт)
- Go API (/api)
- PostgreSQL как хранилище

## Развертывание (в общих чертах)
- Фронт собирается в `dist/` и раздается Nginx
- Бэкенд Go работает как systemd‑сервис
- База подключается по DSN
