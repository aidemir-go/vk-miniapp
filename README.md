VK Mini App для магазина Kotleta Boost (PUBG)

Мини‑приложение ВКонтакте для оформления заказов на буст/сопровождение в PUBG. Проект включает веб‑интерфейс VK Mini Apps и бэкенд на Go с PostgreSQL. Сейчас репозиторий содержит только документацию.

+ Биржа заказов
+ Взятие заказа в работу
+ Чат с заказчиком


**Админ Меню**
+ Сводная статистика
+ Просмотр заказов
+ Просмотр исполнителей
+ Доступ к чатам (read‑only)
+ Добавление массово UC промокодов одним .txt файлом

## Технологии
 Frontend: React + VKUI + Vite
 Backend: Go
 DB: PostgreSQL (Supabase)
 Hosting: VPS + Nginx

 VK Mini App (статичный фронт)
 Go API (/api)
 PostgreSQL как хранилище
